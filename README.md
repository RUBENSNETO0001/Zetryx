# 📝 Zetryx

O **Zetryx** é um sistema digital desenvolvido para otimizar a coleta, processamento e auxílio na documentação de dados estudantis. O projeto visa transformar processos manuais e arquivos de difícil consulta em uma base de dados estruturada e de fácil acesso.

## 🚀 Tecnologias Utilizadas

O projeto utiliza uma stack moderna baseada em JavaScript e Python:

* **Frontend:** React (Vite) - Interface rápida e responsiva.
* **Backend:** Python - Processamento lógico e integração de dados.
* **Banco de Dados:** SQL - Armazenamento estruturado e seguro.

## 🛠️ Funcionalidades Principais

* **Coleta de Dados:** Interface intuitiva para entrada de informações.
* **Processamento Automatizado:** Validação e organização de documentos.
* **Consulta Estruturada:** Facilidade para localizar e reutilizar dados salvos.
* **Interface Minimalista:** Foco na experiência do usuário e usabilidade.

## 🔧 Configuração do Ambiente

Para rodar o projeto localmente, siga os passos abaixo:

### Pré-requisitos
* Node.js e npm/yarn
* Python 3.x
* Ambiente Linux (Ubuntu)

### Instalação

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/zetryx.git](https://github.com/seu-usuario/zetryx.git)
    ```

2.  **Configuração do Backend (Python):**
    ```bash
    cd zetryx/back-end
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    python app.py
    ```

3.  **Configuração do Frontend (React):**
    ```bash
    cd ../front-end
    npm install
    npm run dev
    ```

## 📂 Estrutura do Projeto

* `/back-end`: Scripts em Python e integração com banco de dados SQL.
* `/front-end`: Interface desenvolvida com React e Vite.
